# API-website-registration-part-1

Sistema web de registro de usuarios desarrollado en PHP + MySQL, utilizando una API REST para operaciones CRUD (Crear, Leer, Actualizar y Eliminar).

Este proyecto corresponde a la **Parte 1** del sistema completo:
- Parte 2: Aplicación de escritorio
- Parte 3: Aplicación móvil

Características

- Registro de usuarios
- Validación de datos
- Listado dinámico desde API
- API en PHP con respuestas JSON
- Prevención básica de XSS (`htmlspecialchars`)
- Diseño web con Bootstrap
- Arquitectura separada (Frontend / API)

 Tecnologías utilizadas

- PHP
- MySQL
- Bootstrap 5
- HTML5 / CSS3
- JavaScript
- Git & GitHub
- XAMPP
